# Spec Kit Asset Map

Use this map when a task needs the original GitHub Spec Kit material.

## Root assets

- `.specify/memory/constitution.md`: project constitution and quality gates.
- `.specify/templates/spec-template.md`: feature specification template.
- `.specify/templates/plan-template.md`: implementation plan template.
- `.specify/templates/tasks-template.md`: task breakdown template.
- `.specify/templates/checklist-template.md`: checklist template.
- `.specify/workflows/speckit/workflow.yml`: original workflow metadata.
- `specs/`: feature specifications and derived plans/tasks.
